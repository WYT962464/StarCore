## 🔑 核心机制

### 1. 万物归一
```python
# 输出结果 → 沉淀无极 → 潜能提升
wuji.add_potential(
    amount=0.05,           # 贡献 5% 潜能
    source="cycle_output",
    content={"type": "evolution_result"}
)
```

### 2. 螺旋演化
```python
# 循环次数 → 基础潜能提升 → 演化加速
# 基础潜能 = 初始值 + 循环次数 × 0.05
# 演化速度 = 基础速度 × (1 + 总潜能 × 0.5)

# 第 1 次循环：基础潜能 0.1 → 0.15，速度 1.0 → 1.1x
# 第 10 次循环：基础潜能 0.55，速度 1.0 → 1.8x
# 第 20 次循环：基础潜能 1.0 (上限)，速度 1.0 → 2.5x
```

### 3. 状态流转
```
无极 → 太极 → 两仪 → 四象 → 八卦 → 万物 → 无极 (循环)
  │      │      │      │      │      │
  │      │      │      │      │      └── 万物归一
  │      │      │      │      └── 8 个功能模块
  │      │      │      └── 4 象状态流转
  │      │      └── 2 仪阴阳切换
  │      └── 6 阶段演化
  └── 潜能存储
```

## 🎯 使用示例

```python
from modules.bagua.sixcycle.executor import SixCycleExecutor

# 初始化
executor = SixCycleExecutor()
executor.initialize()

# 执行单次循环
result = executor.execute_full_cycle()
print(f"循环 {result.cycle_number}: 时长 {result.total_duration}s")
print(f"潜能: {result.wuji_potential_before} → {result.wuji_potential_after}")

# 启动自动循环
executor.start_auto_cycle(interval=30)  # 每 30 秒一次

# 获取状态
stats = executor.get_stats()
print(f"总循环: {stats['total_cycles']}, 成功率: {stats['success_rate']}%")
```
