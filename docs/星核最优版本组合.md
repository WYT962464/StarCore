# 星核(StarCore)最优版本组合 v2.0

> 基于实际源码审查 + 服务器文件清单 + 迭代时序，筛选生产级最优组合

## 一、核心客户端(IPA)最优选择

### ✅ 首选：StarCore-v11.ipa (236KB)

**已整合的功能（源码确认）：**
- ✅ 双信道通信（Tweak TCP + MCP HTTP，自动fallback）
- ✅ 文件管理（FileManager沙盒内直写 + Tweak沙盒外写入）
- ✅ 截图（ios-mcp + Tweak双通道）
- ✅ Action执行引擎（44种Tweak action全部支持）
- ✅ 流式/非流式双模式对话
- ✅ Agent Loop（自动多轮执行+反馈闭环）
- ✅ 小智AI集成（Settings页配置token）
- ✅ 多Provider自动切换（火山方舟fallback链）
- ✅ 多模态（图片+文字对话）
- ✅ 沙盒隔离 + 安全防护
- ✅ 下拉刷新 + 路径提示/报告
- ✅ 单文件打包 + 无权限适配 + 防重复输出

**未整合（需评估是否纳入）：**
- ❌ 原生函数调用（NativeFC）— 独立模块，v11通过Tweak shell替代

## 二、系统级越狱插件(DEB)最优选择

### ✅ 首选：StarCoreTweak-v9.2.deb (64KB)

**已修复的关键bug：**
- v8.4→v8.5：文件操作增强（readFile/writeFile/listFiles）
- v8.5→v9.0：内存优化+后台驻留稳定性
- v9.0→v9.1：frontmostApp获取修复+按键卡死修复+AX fallback
- v9.1→v9.2：**主线程死锁修复**（sendButtonEvent改用global queue）+ **SBSCopy崩溃移除**（改用SBAppSwitcherModel）

**完整action列表（44个）：**
ping, tap, swipe, longPress, doubleTap, dragAndDrop, pressHome, pressPower, pressVolumeUp, pressVolumeDown, toggleMute, wakeAndHome, keyPress, textInput, inputText, typeText, screenshot, getScreenInfo, getUIElements, getElementAtPoint, getFrontmostApp, listApps, listRunningApps, openApp, killApp, installApp, uninstallApp, openURL, getDeviceInfo, getBrightness, setBrightness, getVolume, setVolume, shell, setClipboard, getClipboard, readFile, writeFile, appendFile, listFiles, getScreenSize, initDevice, diagnose, validate

**兼容性：** iOS 11-16 全系列越狱设备 | 与所有版本IPA兼容

## 五、黄金组合（一键部署）

```bash
# 1. 安装Tweak
curl -k -o /tmp/sc.deb <SERVER>/StarCoreTweak-v9.2.deb
dpkg -i /tmp/sc.deb && killall SpringBoard

# 2. 安装App（respring后）
curl -k -o /tmp/sc.ipa <SERVER>/StarCore-v11.ipa
# TrollStore/Filza安装

# 3. 启动poller
RELAY_URL='<SERVER>' python3.12 /var/mobile/StarCore/phone_poll.py
```

## 六、部署铁律

1. **v11 + v9.2 = 95%+场景稳定运行**，优先用此组合
2. 遇bug先找专项修复版，不降级主程序
3. 小智必须用v2代理+token更新脚本
4. 所有脚本用绝对路径执行
5. v9.2已修死锁和SBSCopy崩溃，不要回退到v9.1或更早
