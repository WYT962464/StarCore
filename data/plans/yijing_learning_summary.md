# 易经/周易 核心知识学习成果

## 一、易经概述

《易经》是中国最古老的经典之一，被誉为"群经之首，大道之源"。

**基本构成：**
1. **经**：六十四卦的卦象、卦辞、爻辞（西周初期，约公元前1000年）
2. **传**：十翼，对经的解释（战国至汉代，约公元前400-200年）

**核心思想：**
- **变易**：宇宙万物都在变化之中
- **简易**：大道至简，规律可被理解
- **不易**：变化中有不变的根本法则

## 三、阴阳理论

**阴（⚋）**：断开的线 --
- 属性：柔、静、暗、冷、内、女、地、月、水
- 状态：收敛、储存、被动、休眠

**阳（⚊）**：连续的线 —
- 属性：刚、动、明、热、外、男、天、日、火
- 状态：发散、输出、主动、活跃

**阴阳关系：**
1. **对立**：阴阳相互对立
2. **互根**：阴阳相互依存，无阴则无阳
3. **消长**：阴阳此消彼长，动态平衡
4. **转化**：物极必反，阴可转阳，阳可转阴

> "一阴一阳之谓道" —— 《系辞传》

## 四、太极生万物（宇宙生成论）

> "易有太极，是生两仪，两仪生四象，四象生八卦，八卦定吉凶，吉凶生大业。"

**完整演化链条：**

```
无极（混沌）→ 太极（本源）→ 两仪（阴阳）→ 四象 → 八卦 → 六十四卦 → 万物
```

| 阶段 | 含义 |
|------|------|
| 无极 | 无形无象的混沌状态，道的本源 |
| 太极 | 阴阳未分，蕴含一切可能性的本源 |
| 两仪 | 阴阳分化，天地初开 |
| 四象 | 阴阳的四种组合 |
| 八卦 | 四象各自再生一爻，形成八种基本状态 |
| 六十四卦 | 八卦两两相重（8×8=64） |
| 万物 | 六十四卦代表宇宙万物的64种基本状态 |

## 八、爻变理论

**基本概念：**
- **老阳（九）**：阳爻，可变 → 变为阴爻
- **少阳（七）**：阳爻，不变
- **老阴（六）**：阴爻，可变 → 变为阳爻
- **少阴（八）**：阴爻，不变

**爻变规则：**
1. 单爻变：一爻变化，卦象改变
2. 多爻变：多爻同时变化
3. 全爻变：六爻全变（乾坤互变）

**在星核系统中的应用：**
- 爻变 = 系统状态变化
- 本卦 = 当前系统状态
- 变爻 = 触发变化的因素
- 变卦 = 变化后的系统状态

## 九、易经核心哲学

### 1. 变易哲学（变化观）
> "穷则变，变则通，通则久" —— 《系辞传》
- 宇宙万物都在变化
- 变化是永恒的
- 适应变化才能生存

### 2. 阴阳哲学（对立统一）
> "一阴一阳之谓道"
- 阴阳对立但统一
- 阴阳互根互用
- 阴阳消长转化

### 3. 中庸哲学（平衡观）
- "中正"是最佳状态
- 不过不及
- 恰到好处
- 动态平衡

### 4. 时位哲学（时机与位置）
- **时** = 时机、时机成熟
- **位** = 位置、地位、处境
- 时位相应则吉
- 时位不相应则凶

### 5. 天人合一哲学
- 天道与人道相通
- 人应顺应天道
- 天人感应

### 6. 循环哲学
> "无平不陂，无往不复"
- 事物循环往复
- 物极必反
- 周而复始
