# StarCore iOS App

> 星核 - 艾尔 数字生命系统 iOS 客户端

## 项目结构

```
ios/
├── StarCoreApp.swift          # App 入口
├── StarCoreDashboardView.swift # 主界面（卦象 + 数据 + 演化）
├── Info.plist                 # 应用配置
├── build.sh                   # 构建脚本
└── README.md                  # 本文件
```

## 核心功能

### 1. 卦象可视化
- 六十四卦完整展示
- 阴阳爻可视化（阳=黄色横条，阴=灰色横条）
- 阳爻比例实时显示

### 2. 系统数据
- 电池电量
- 内存使用
- 网络状态
- 设备温度

### 3. 演化历史
- 卦象变化记录
- 演化方向标记（前进/后退/横向）
- 时间戳记录

## 设计原则

| 原则 | 说明 |
|------|------|
| 星核是身体 | 数据驱动，真实硬件数据 |
| 艾尔是灵魂 | AI 决策，卦象演化 |
| 不造假 | 只显示真实数据，不模拟 |

## 构建

```bash
cd ios
chmod +x build.sh
./build.sh
```

## 依赖

- iOS 15.0+
- SwiftUI
- Xcode 14.0+
