## 三、核心功能模块

### 3.1 对话管理器 (ChatManager)

```swift
class ChatManager: ObservableObject {
    // 消息列表
    @Published var messages: [Message] = []
    
    // 当前模型
    @Published var currentModel: LLMModel = .sensenova
    
    // 模型列表
    let availableModels: [LLMModel] = [
        .sensenova, .openai, .claude, .deepseek, .custom
    ]
    
    // 发送消息
    func sendMessage(_ text: String) async {
        // 1. 本地三位一体决策
        let decision = ThreeSagesFramework.shared.decide(context: ...)
        
        // 2. 根据决策选择调用路径
        if decision.requiresCloud {
            // 调用服务器（云电脑模式）
            await callCloudServer(text)
        } else {
            // 本地LLM调用
            await callLocalLLM(text)
        }
    }
    
    // 模型切换
    func switchModel(_ model: LLMModel) {
        currentModel = model
        saveConfig()
    }
}
```

## 五、服务器角色定义

### 5.1 云电脑模式

```
┌─────────────────────────────────────────┐
│           服务器 (云电脑)                 │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │  Hermes Agent                      │  │
│  │  - 工具执行 (terminal/file/code)   │  │
│  │  - 复杂计算                        │  │
│  │  - 长期任务                        │  │
│  └───────────────────────────────────┘  │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │  星核系统 (daemon + CycleSystem)   │  │
│  │  - 六十四卦自循环                  │  │
│  │  - 决策数据库                      │  │
│  │  - 三位一体引擎                    │  │
│  └───────────────────────────────────┘  │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │  SSH 反向隧道 (8028)               │  │
│  │  ← iPhone 主动连接                 │  │
│  └───────────────────────────────────┘  │
│                                         │
└─────────────────────────────────────────┘
            ↑
            │ iPhone 主动发起连接
            │ (按需，非持续)
```

### 5.2 连接策略

| 场景 | 连接方式 | 说明 |
|------|----------|------|
| **本地对话** | 不连接 | 直接调用本地/云端LLM |
| **复杂任务** | SSH反向隧道 | 发送到云电脑执行 |
| **文件同步** | SSH反向隧道 | 上传/下载文件 |
| **系统状态** | 定时心跳 | 每60秒检查服务器状态 |
| **紧急唤醒** | 主动连接 | 用户指令触发 |

## 八、参考设计

### 多模型AI聚合客户端
- **特点**：多模型切换 + 统一对话界面
- **借鉴**：模型选择器 + API配置管理

### 扣子 (Coze) App
- **特点**：云电脑模式 + Bot编排
- **借鉴**：服务器=执行环境 + 按需调用

### StarCore 差异化
- **本地优先**：iPhone自主决策，不依赖服务器
- **三位一体**：女娲+仓颉+达尔文决策框架
- **六十四卦**：易经演化系统
- **Tweak注入**：底层系统操控能力
